Muziekpersoonlijkheidsquiz (Fullstack webapplicatie)

Technologieën: HTML, CSS, JavaScript (ES6 modules), Node.js, Express, SQLite, Fetch API, Figma, Git

Rol: Fullstack developer

  • Een interactieve quiz waarin gebruikers vragen beantwoorden om hun muziekpersoonlijkheid te ontdekken en op basis daarvan een gepersonaliseerde line-up krijgen
  • Backend gebouwd met Node.js en Express, met SQLite als database.
  • Frontend dynamisch opgebouwd met JavaScript, modules en Fetch API.
  • Data en gebruikersinteractie worden realtime verwerkt en gepresenteerd.

Interactieve receptenzoeker (Frontend one-page app)

Technologieën:  HTML, CSS, JavaScript, Fetch API, externe API (TheMealDB), LocalStorage

Rol: Frontend developer

  • Gebruikt live data via een externe API, en slaat favorieten lokaal op via LocalStorage.
